Високоякісне синтетичне моторне масло для сучасних двигунів. Забезпечує відмінний захист двигуна при будь-яких температурах.

  • Повністю синтетичне
  • API SN/CF
  • ACEA A3/B4
  • Об'єм: 5л

Повний опис товару

Моторне масло Mobil Super 3000 X1 5W-40 - це повністю синтетичне моторне масло преміум-класу, розроблене для забезпечення видатної продуктивності та захисту сучасних бензинових і дизельних двигунів у всіх умовах експлуатації.

Переваги:

  • Відмінний захист двигуна при високих температурах
  • Швидке прокачування при низьких температурах
  • Зменшення тертя та зносу деталей двигуна
  • Захист від нагароутворення та шламу
  • Оптимальна робота каталітичного нейтралізатора

Рекомендовано для більшості сучасних автомобілів європейських, американських та азійських виробників.
